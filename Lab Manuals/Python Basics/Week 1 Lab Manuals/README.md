## 🗺️ Week Overview

Week 1 is your entry point into Python. You will write your very first Python program, understand what an interpreted language is, learn how to display output, and get comfortable with the tools every Python programmer uses every day.

By the end of this week, you won't just know *that* Python works — you'll understand *how* it works, and why it's one of the most popular programming languages on Earth.

---

## 🎯 Learning Objectives

By the end of Week 1, you will be able to:

- [ ] Open and use the Python interactive shell (REPL)
- [ ] Create, save, and run a Python script file (`.py`)
- [ ] Use `print()` to display text, numbers, and booleans
- [ ] Control `print()` behavior using `sep` and `end` parameters
- [ ] Write single-line and inline comments using `#`
- [ ] Explain the difference between interpreted and compiled languages
- [ ] Describe what the REPL is and when to use it

## 💡 Study Tips for Week 1

- **Do not skip the REPL section.** The interactive shell is your fastest learning tool throughout this course.
- **Type every code example.** Don't copy-paste — your fingers need to learn the syntax too.
- **Experiment freely.** Change values in the examples. What happens if you put a number inside quotes? Try it.
- **Read error messages.** When Python gives you an error, it's telling you exactly what went wrong. Learn to read them.
