1. Created virtual enviroment using venv package.
cmd: python -m venv [enviroment  name]
cmd: python -m venv test-venv
2. To active venv using [path_to_Scripts_folder]\activate.bat
3. To deactivate, we dont need to goto scirpts folder, just hit deactivate cmd.
4. pip helps us to add new packages to the global or venv
cmd: pip install django==5.2.15
5. To list installed packages in venv, we can use pip list
6. To create django project use: django-admin startproject [project_name]
7. To start the project we need to call runserver command with help of manage.py 
cmd: python manage.py runserver
8. To create first user we can use createsuperuser command same like runserver after applying migrations
	a. python manage.py makemigrations
	b. python manage.py migrate
	c. python manage.py createsuperuser # admin / admin
